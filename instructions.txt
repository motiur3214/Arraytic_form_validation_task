# Project Setup Instructions

1. **MySQL Database Setup**:
   - Import the `schema.sql` file from sql folder to DB.
   - Import the `dummy_data.sql` from sql folder for dummy data

2. **Project Requirements**:
   - PHP 8.1.
   - XAMPP/WAMP/LARAGON.

2. **Project Setup**:
   - Clone from GitHub or Place the `Arraytic_form_validation_task` directory inside the `htdocs` (or `www`) directory of your local server (e.g., XAMPP, WAMP).
   - Rename example.env to .env and give db information
   - Start your server.

3. **Accessing the Project**:
   - Form Page: http://localhost/my_php_project/public/
   - Report Page: http://localhost/my_php_project/public/report.php

4. **Additional Notes**:
   - Ensure you have the necessary PHP extensions enabled (e.g., PDO for MySQL).
   - The project was tested in a local environment using LARAGON.
